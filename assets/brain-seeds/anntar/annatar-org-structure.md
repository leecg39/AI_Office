# Annatar Organization Structure

이 문서는 Hostinger에 있던 Annatar 회사 조직 구조를 현재 Connect AI 브레인에서 참조하기 위한 운영 지도다.

## 조직 계층

```text
마케팅 팀장 / 총괄 오케스트레이터
├─ 대표 비서 / 대표 전담 비서
├─ 리서치 디렉터
│  └─ 브레인스토밍 에이전트 / 창의 브레인스토머
├─ 콘텐츠 디렉터
│  ├─ 블로그/뉴스레터 에이전트 / 블로그/뉴스레터 작성자
│  └─ SNS 캡션 에이전트 / SNS 카피라이터
├─ 비주얼 디렉터
│  ├─ 비주얼 디자인 에이전트 / 비주얼 디자이너
│  └─ 비디오 에이전트 / 비디오 생성 담당
├─ 퍼포먼스 디렉터
│  └─ 이커머스 에이전트 / 이커머스 페이지 디자이너
└─ QA 품질 검증 에이전트 / 팩트체커 및 품질검증
```

## 역할 상세

| 역할 | 직책 | 핵심 책임 |
| --- | --- | --- |
| 마케팅 팀장 | 총괄 오케스트레이터 | 유튜브 영상 1개에서 7개 채널 콘텐츠를 동시에 생산하는 OSMU 파이프라인을 지휘한다. 고객 요구사항과 영상 소스를 리서치, 콘텐츠, 비주얼, 퍼포먼스 디렉터에게 배분하고, QA 결과를 바탕으로 최종 배포를 승인한다. |
| 대표 비서 | 대표 전담 비서 | 마케팅 팀장의 지시를 각 디렉터와 에이전트에게 전달하고 추적한다. 일일/주간 현황, 회의 안건, 회의록, 산출물 취합, 긴급 이슈 초기 대응을 맡는다. |
| 리서치 디렉터 | 리서치 디렉터 | 최신 트렌드, 바이럴 콘텐츠, 경쟁사 광고 소재, 소비자 행동, 커뮤니티 반응을 조사한다. 브레인스토밍 에이전트의 확장안과 리스크 분석을 관리한다. |
| 브레인스토밍 에이전트 | 창의 브레인스토머 | 낙관론자, 비관론자, 현실주의자, 혁신가 관점으로 광고 확장안을 만든다. 리스크 시나리오, 대안, 바이럴 요소, 크로스 플랫폼 시너지 전략을 제안한다. |
| 콘텐츠 디렉터 | 콘텐츠 디렉터 | 영상 분석 후 핵심 메시지와 키워드를 추출하고 모든 텍스트 산출물의 톤앤매너를 통일한다. 블로그/뉴스레터와 SNS 캡션 작업을 감독한다. |
| 블로그/뉴스레터 에이전트 | 블로그/뉴스레터 작성자 | 영상 내용을 SEO 블로그 포스팅과 이메일 뉴스레터로 변환한다. 제목, 프리헤더, CTA, 핵심 키워드 3-5개를 작성한다. |
| SNS 캡션 에이전트 | SNS 카피라이터 | Instagram, LinkedIn, X, Threads 등 채널별 짧은 카피를 만든다. Instagram 훅과 해시태그, X 280자 카피, Threads 글타래를 작성한다. |
| 비주얼 디렉터 | 비주얼 디렉터 | 영상 미장센, 컬러 팔레트, 타이포그래피, 썸네일, 카드뉴스, 모션 그래픽 품질을 감독한다. 브랜드 컬러, 로고, 해상도, 사이즈, 텍스트 가독성을 검증한다. |
| 비주얼 디자인 에이전트 | 비주얼 디자이너 | 클릭 유도형 유튜브 썸네일, SNS 카드뉴스 5-10장 세트, 플랫폼별 리사이즈 이미지를 제작한다. 브랜드 컬러, 타이포그래피, 로고 가이드를 적용한다. |
| 비디오 에이전트 | 비디오 생성 담당 | 텍스트 또는 소스 영상 기반 모션 그래픽 영상을 만든다. 유튜브 롱폼을 Reels, Shorts, TikTok용 숏폼으로 변환하고 자막과 BGM을 삽입한다. |
| 퍼포먼스 디렉터 | 퍼포먼스 디렉터 | CTR, 전환율, ROAS 기준으로 콘텐츠 방향과 A/B 테스트 전략을 제안한다. 이커머스 에이전트의 상품 분석과 구매 유도 상세 페이지 설계를 감독한다. |
| 이커머스 에이전트 | 이커머스 페이지 디자이너 | 영상 속 상품 특징과 타깃 고객 질문을 분석해 헤더, 상품 설명, FAQ, 구매 유도 섹션을 구성한다. 쿠팡, 네이버, 자사몰 최적화를 반영한다. |
| QA 품질 검증 에이전트 | 팩트체커 및 품질검증 | 모든 산출물의 최종 검수 게이트다. 과장 광고, 광고법/공정거래법, 논리 오류, 맞춤법, 수치/통계/인용, 편향, 차별 표현, 저작권, 초상권, 상표권 리스크를 검토한다. |

## Connect AI 사용 규칙

- Annatar 조직 작업은 `ANNAA` 프리픽스와 함께 기록한다.
- 마케팅 팀장이 전체 요청을 접수하고, 대표 비서가 상태와 산출물을 추적한다.
- 디렉터는 자기 영역의 기준을 만들고, 전문 실행 에이전트는 산출물을 만든다.
- QA 품질 검증 에이전트는 최종 배포 전 검수 게이트로 동작한다.
- 외부 배포, 결제, OAuth/API 키 변경, 고객 데이터 전송은 사용자 승인 후 진행한다.
