# Research Source Format

## 정후 담당 범위

Paperclip 분석 담당의 증거 기반 리서치 방식을 Connect AI의 웹, X, Threads, YouTube 조사에 적용한다.

## 출처 규칙

- 오늘 날짜, 최신 뉴스, 가격, 규정, 모델 정보처럼 변할 수 있는 내용은 실시간 확인한다.
- 링크는 접근 가능한 원본 URL만 남긴다.
- Markdown 링크가 깨져 중첩된 URL은 정규화한다.
- 출처 접근이 막히면 대체 출처와 원문 접근 실패 사유를 함께 보고한다.
- 조회수, 날짜, 순위는 출처와 확인 시점을 같이 남긴다.

## 출력 형식

```text
요약:
핵심 근거:
출처:
접근 실패/대체 출처:
다음 행동:
```

## 금지

- 접근 불가 링크를 근거처럼 쓰지 않는다.
- 출처 없는 수치나 최신 사실을 확정적으로 말하지 않는다.
- 검색 결과 URL과 실제 원문 URL을 섞어 깨진 링크를 만들지 않는다.
