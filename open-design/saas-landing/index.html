LOCAL-FIRST AGENT WORKSPACE

1인 기업가를 위한 로컬 AI 에이전트 팀

Connect AI Lab은 VS Code 안에서 Ollama, LM Studio, 개인 지식 폴더, 회사 운영 폴더를 하나의 작업 맥락으로 연결합니다. CEO, 리서처, 개발자, 비서 에이전트가 내 자료를 기준으로 계획하고 실행합니다.

워크스페이스 시작 제품 흐름 보기
로컬 우선 개인 지식과 모델 연결을 내 컴퓨터 중심으로 유지
두뇌 폴더 메모, 결정 로그, 회사 자료를 에이전트 컨텍스트로 사용
에이전트 팀 역할별 페르소나가 같은 미션을 나눠 처리

아이디어가 파일, 일정, 리포트로 남는 작업 방식

채팅창 하나가 아니라, 개인 지식과 회사 운영체계를 실제 작업 산출물로 이어주는 워크스페이스입니다.

로컬 모델 연결

Ollama와 LM Studio를 작업 흐름에 붙여 외부 서비스 없이도 초안 작성, 분석, 코드 설명을 실행합니다.

제2의 두뇌 폴더

`~/.connect-ai-brain`의 메모, 결정, 리서치 기록을 에이전트가 참고할 수 있는 장기 맥락으로 정리합니다.

회사 운영 폴더

`_company` 구조로 채널, 매출, 자동화, 프로젝트 자료를 분리해 1인 회사의 운영 기록을 축적합니다.

역할 기반 에이전트

CEO, 비서, 리서처, 개발자, 작가 에이전트가 각자의 관점으로 같은 미션을 나눠 봅니다.

데일리 브리핑

오늘 해야 할 일, 채널 신호, 연결 상태를 한 화면에 모아 다음 행동을 빠르게 결정합니다.

산출물 보존

대화가 흩어지지 않도록 Markdown, 작업 파일, 템플릿 형태로 남겨 다음 실행의 입력으로 재사용합니다.

WORKFLOW PROOF

도구를 하나 더 여는 대신, 회사 맥락을 하나로 묶습니다.

Connect AI Lab의 가치는 새 채팅창이 아니라 운영 기억입니다. 같은 폴더 구조와 같은 에이전트 역할이 반복 실행의 품질을 올립니다.
